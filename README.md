# NASA-OCO2-Satellite-CO2-Analysis
Satellite-based atmospheric CO₂ analysis using NASA OCO-2 spaceborne data. Processes 500MB+ HDF5/NetCDF4 granules to quantify a global +13.6 ppm CO₂ rise (2019–2025) and performs inter-product validation between two NASA retrieval algorithms over India. Built in MATLAB with geospatial filtering and multi-temporal visualization.
